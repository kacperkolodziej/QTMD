Basic usage

To start talking you need to connect to server. At the moment the only working server for users is running under domain talk.tmnd.net and port number 5000. It uses fully verified SSL certificate. To connect, click Connect button. You can track connection progress at the bottom of main application's window (at status bar).

When you are connected to server you can talk in main room. On the right you can see two lists. First from top is list of available public rooms on server you are connected to. The list below that contains all connected users.

To enter to room you can use /room command or double click on room name on the list. If you want to have private conversation with somebody, double click his name on the list of users.

To leave room or private conversation you can type /leave command or double click on tab.
